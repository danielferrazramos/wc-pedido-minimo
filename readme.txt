Plugin para configurar valor mínimo para finalização de pedidos no WooCommerce.

== Description ==

Defina um valor mínimo para a finalização de pedidos no WooCommerce.

Caso o cliente tente finalizar a compra sem atingir o valor mínimo, será exibida mensagem informando o valor mínimo, o valor total do pedido e o valor necessário para atingir o mínimo configurado para a loja.

* É necessário que o WooCommerce esteja instalado e ativo para utilizar este plugin.

Description: Plugin to set minimum value for order checkout in WooCommerce.


== Installation ==

= Instalação do plugin: =

- Faça o upload da pasta wc-pedido-minimo para a pasta de plugins ou instale diretamente através da sua instalação do Wordpress .
- Ative o plugin.


= Configuração do plugin: =

- Acesse WooCommerce->Configurações e acesse a guia "WC Pedido Mínimo".

- Acesse "WooCommerce->Configurações" e clique na guia "WC Pedido Mínimo".

- Preencha com o valor desejado e clique em "Salvar Alterações".
